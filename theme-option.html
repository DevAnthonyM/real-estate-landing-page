Real-Estate- Theme Options

Learn about useful options

Logo And Favicon

After installing Option Tree plugin, Theme Options can be found in Appearance section. In this part, there are a lot of options that help you to build your site easily.

Go to Appearance --> Theme Options and find needed options

With Logo & Favicon Option, you can insert logo of your site which are either image or text. You can also add image for footer logo. Look at our examples of logo image. Make your site more professional by enabling favicon.

Theme Style

You can choose your theme style here: Default Color or Single Image

Header Top Option

With this option, you can insert your contact information nad insert your social networks including Email and Phone number. Also, you can choose to show or hide language switcher in your header.

Option Breadcrumb

To use this option, Breadcumb NavXT plugin must be installed. This option allows you to show page navigation and make it easy to manage your site.

Breadcrumb Option

Footer Options

This option is used to configure footer of Blog, Portfolio and Shop. You can also configure Footer in Theme Options

Step 1: Go to Admin----> Appearance---> Theme Options---> Footer Options

Step 2: You can configure footer. Besides, there are 3 options to configure for Footer 1, Footer 2, Footer3

Footer Option
Footer

You should also refer to Footer Settings for instructions

404 Page

This option allows you to create a meaningful 404 page instead of boring one.

Step 1: Go to Appearance --> Theme Option to open this option.

Step 2: Click 404 Page

Font Option

Step 1: Go to Appearance --> Theme Option

Step 2: Choose Font Option

You can choose more fonts by clicking Google Fonts. You can follow step by step as following to insert new font.

  • Step 2.1 - Choose new font you want
  • Step 2.2 - Click Add to Collection --> Use, copy code and font family, then paste to Font Option in Theme Option.
  • Step 2.3 - You should add positions you want to change in Selector box These positions can be a div or a class...

Custom CSS

This option allows you to add some extra configurations such as changing color, font size, etc by using CSS codes.

Step 1: Go to Appearance --> Theme Option

Step 2: Choose Custom CSS

Partner Option

This option allows you to configure options in Partner section

Step 1: Go to Appearance --> Theme Option

Step 2: Choose Partner

Blog options

Step 1: Go to Appearance --> Theme Option

Step 2: Choose Blog Options

In this option, select a blog style which can be List or Grid and configure other options.

Single Listing

This option allows you to configure options in your single blog.

Agent Directory

This option allows you to configure options in agent page.

Agent Detail

This option allows you to configure options in agent page.

Property All

This option allows you to configure options in agent page.

Property Detail

This option allows you to configure options in agent page.

Property Gallery

This option allows you to configure options in agent page.

General Options

This option allows you to configure options in your pages.
