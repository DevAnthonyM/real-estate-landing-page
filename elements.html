Frequent Elements

This is a general instruction of some frequent elements used to make up your website. Each element is set default value of padding is 50px. To offset this padding, you can open Row Setting and set up padding again with new value is 0px or other value.

Header Option

In this element,you can set up your logo types and select menu to show.

In addition, you can also configure color for Logo Text

NOTE: Settings in this element are applied for the page including this element only. In case you want to change logo and header type for other Default page, please go to:

Logo: Go to Appearance > Theme Options > Logo & Favicon

Header Type: Go to Appearance > Theme Options > Header Options

Slider

You can use this element to add sliders on your page. This element includes the following options:

  • Images: Select and upload your slide images.
  • Title: Enter Slide title
  • Description: Enter the slide description
  • Font Color: Select or enter a color code
  • Height Slider: Show the slide images in full height or auto height.
  • Autoplay: Select to play the slides automatically or manually.
  • Speed slider: Enter the time of slide transition. Eg: 5000 seconds.
  • Show Dots: Show/Hide Dots on the slides.
  • Show/Hide Overlay: Show/Hide the overlay on the slides
