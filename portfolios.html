Properties Page

Learn how to create demo properties versions!

Properties Grid

How to create:

You can refer to Blog Posts - Portfolios for more guidance of creating portfolio. Create set of your articles with categories.

To create this Property Grid version, simply, you just need to do this following steps.

Step1: Go to Pages---->Add New

Step2: Select Default Template as its template

Step3: Add shortcode as image

You can also refer to Theme Option --> Single Portfolio Option to set up your portfolio.

Properties Detail

How to create:

You can refer to Blog Posts - Portfolios for more guidance of creating portfolio. Create set of your articles with categories.

To create this property detail, you can go to Menus and add portfolio for Properties Detail.
